# Contributing to Keraline OS

Thank you for your interest in contributing to Keraline OS.

Keraline OS is an open-source operating system written primarily in C and x86 Assembly. The project is currently in early development and all contributions are welcome.

## Before Contributing

Please:

- Read the README.md
- Search existing issues before creating a new one
- Keep pull requests focused on a single change
- Test your changes whenever possible

## Ways To Contribute

### Bug Fixes

If you find a bug:

1. Open an issue.
2. Explain the problem.
3. Describe how to reproduce it.
4. Include logs or screenshots if available.

### Features

If you want to add a feature:

1. Open an issue first.
2. Explain the feature.
3. Wait for approval before starting large changes.

### Documentation

Documentation improvements are always welcome.

Examples:

- README improvements
- Developer guides
- Build instructions
- Code comments
